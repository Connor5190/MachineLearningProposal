Machine Learning Proposal

Literature Review

Louppe provides a theoretical and practical analysis of Random Forests, explaining how ensemble learning reduces variance and how impurity-based feature importance is computed [1], supporting our use of Random Forest and guiding interpretation of influential attributes. Strobl et al. show that standard Random Forest importance measures can be biased toward certain predictor types, especially when variables differ in scale or category count [2], which is important given our mixed feature types. He and Choi demonstrate that ensemble machine learning methods for sports outcome prediction can achieve strong predictive performance while maintaining interpretability [3], reinforcing the suitability of ensemble models like Random Forest for our task.

Dataset Description

The UFC Dataset is a comprehensive MMA dataset containing every recorded UFC fight along with detailed fighter statistics.

Dataset: Kaggle — UFC Dataset 2024

The dataset includes:

  • Complete fight history for all UFC events
  • Fight-level statistics for both competitors in each bout
  • Fighter performance metrics at the time of each fight (e.g., strikes landed, takedowns, submissions, control time, etc.)
  • Event metadata such as event name and date
  • A current roster of UFC fighters
  • Updated fighter profiles with their physical attributes and career records
  • Aggregated career statistics reflecting their most recent information

This will allows us to analyze:

  • How pre-fight statistics relate to fight outcomes
  • Predictive modeling using stats available at the time of the fight
  • Current fighter comparisons based on up-to-date profile data

Because it includes both historical fight-level data and a current snapshot of fighter profiles, the dataset supports the analysis we are looking to achieve.

Problem Statement

Mixed martial arts outcomes are influenced by a combination of fighter attributes, historical performance metrics, and matchup dynamics. Given a dataset containing every UFC fight along with each fighter's statistics at the time of their bout, the objective is to develop a predictive model that determines the probability of a fighter winning a future matchup.

Specifically, this project aims to:

  • Engineer features that reflect performance trends and stylistic advantages
  • Train and evaluate a supervised machine learning model
  • Predict the likely winner of an upcoming UFC fight based solely on information available before the fight occurs

The main challenge is to accurately model fight outcomes using only pre-fight data, ensuring the system reflects realistic, real-world prediction constraints.

Motivation

We're a group of sports fans who regularly watch UFC events together. Half the fun is debating the fights before they start: who has the better striking, whose grappling will hold up, and whether reach or experience will make the difference. We’re always making predictions and arguing our cases like professionals. This project is exciting because it adds a data-driven layer to something we already love. Instead of going off gut feeling or reputation, we want to see if the numbers back up our predictions. Can striking accuracy predict outcomes? Does reach matter as much as we think? Do certain styles consistently win? By building a model using real UFC data, we’re putting our fight-night debates to the test and combining our interest in sports with machine learning in a way that’s both analytical and fun.

Data Preprocessing

We will address missing values with scikit-learn's SimpleImputer, dropping features with many missing fields and imputing medians for essential metrics to preserve sample integrity. We will use pandas.DataFrame operations for feature engineering, transforming the data into pairwise matchups and calculating relative differentials such as striking volume, takedown defense, and reach. To mitigate positional bias, we will symmetrize the data by duplicating each fight, swapping fighters, multiplying differential features by -1, and inverting the target variable. We will normalize variables with widely different ranges using StandardScaler for consistency in training.

Machine Learning Algorithms

We will use three supervised classification algorithms to predict a binary outcome. LogisticRegression will provide an interpretable baseline. RandomForestClassifier will train multiple decision trees on different subsets of data, using majority voting to predict the winner. Our main model, XGBClassifier, builds trees sequentially to correct prior errors and applies L1 and L2 regularization to control complexity and improve performance. We apply supervised learning for predictive analytics, using linear classification as a baseline and tree-based models for final predictions

Results and Discussion

To evaluate our prediction model, we will use quantitative metrics for various targets: binary outcomes, probabilistic odds, and specific fight endings.

  • Binary Winner/Loser: We will evaluate win/loss predictions using accuracy (Total Correct / Total Predictions). This simple metric provides an effective baseline to ensure the model functions correctly.
  • Winning Probability Projections: We will use Brier Score and Log Loss to evaluate model calibration by comparing projected probabilities to the ground truth. Brier Score measures the mean squared difference from the true outcome, while Log Loss heavily penalizes overconfident, incorrect predictions. We will benchmark these scores against official closing odds.
  • Method and Timing of Victory: If a fight ends via knockout or submission, we will apply Mean Absolute Error to the predicted fight duration. For decisions, we will use Root Mean Squared Error to measure the variance between our projected and the true scorecards.

Our primary goal in this project is to develop a UFC prediction model that performs comparable to betting market favorites. In this project, we will utilize tree models like XGBoost, Random Forest, and Logistic Regression. These algorithms require significantly less computing power than other models like neural nets, ensuring we remain environmentally conscious.

We expect that we will achieve an accuracy of at least 60%, which would be an improvement over a random guess baseline. Although this is below the 65%-68% win rate of Vegas favorites, we believe it’s still competitive for our model relying exclusively on publicly available fighting metrics and match stats.
