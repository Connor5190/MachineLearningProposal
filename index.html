Machine Learning Proposal

Dataset Description

The UFC Dataset is a comprehensive mixed martial arts dataset containing every recorded UFC fight along with detailed fighter statistics.

Dataset: Kaggle — UFC Dataset 2024

The dataset includes:

  • Complete fight history for all UFC events
  • Fight-level statistics for both competitors in each bout
  • Fighter performance metrics at the time of each fight (e.g., strikes landed, takedowns, submissions, control time, etc.)
  • Event metadata such as event name and date

In addition to historical fight data, the dataset also contains:

  • A current roster of UFC fighters
  • Updated fighter profiles with their latest physical attributes and career records
  • Aggregated career statistics reflecting their most recent information

This structure allows users to analyze:

  • How fighter performance evolves over time
  • How pre-fight statistics relate to fight outcomes
  • Predictive modeling using stats available at the time of the fight
  • Current fighter comparisons based on up-to-date profile data

Because it includes both historical fight-level data (time-specific stats) and a current snapshot of fighter profiles, the dataset supports longitudinal analysis as well as modern fighter evaluation and modeling.

Problem Statement

Mixed martial arts outcomes are influenced by a combination of fighter attributes, historical performance metrics, and matchup dynamics. Given a dataset containing every UFC fight along with each fighter's statistics at the time of their bout, the objective is to develop a predictive model that determines the probability of a fighter winning a future matchup.

Specifically, this project aims to:

  • Utilize historical fight data and pre-fight fighter statistics
  • Engineer features that reflect performance trends and stylistic advantages
  • Train and evaluate a supervised machine learning model
  • Predict the likely winner of an upcoming UFC fight based solely on information available before the fight occurs

The core challenge is to accurately model fight outcomes using only pre-fight data, ensuring the system reflects realistic, real-world prediction constraints.

Motivation

We're a group of big sports fans and we regularly watch UFC events together. Half the fun for us is debating matchups before the fights start: who has the better striking, whose wrestling will hold up, whether reach or experience will make the difference, etc. We're always making predictions and arguing our cases like analysts.

This project is exciting for us because it takes something we already love doing and adds a data-driven layer to it. Instead of just going off gut feeling or fighter reputation, we want to see if the numbers actually back up our predictions. Can striking accuracy really predict outcomes? Does reach matter as much as we think? Do certain styles consistently win?

By building a model using real UFC fight data, we're essentially putting our fight-night debates to the test. It's a way to combine our interest in sports with machine learning and see if we can build something that's both analytical and fun.
